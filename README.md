### Get started
- choose editor of your choice
- install python 3.x (i guess)
- create and open project directory
- Clone repo `git clone https://github.com/PeterSteiglechner/workshop_mathmodels.git` 
- open terminal
- Install packages `pip3 install -r requirements.txt`
- Enjoy and learn
